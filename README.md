## 1. THE ORIGIN & ABSTRACT
The Volition Coalition (TVC) is a Multi-Agent Context-Sharded Consensus (MACSC) framework designed for total operator sovereignty (Phi). Born on the glass of a mobile terminal, TVC democratizes elite-level strategic ideation. It replaces automated AI bias with an Architect-Governed Forge, providing individuals—regardless of their formal discipline or field experience—the power of a world-class think tank.

## 2. CORE TECHNICAL PILLARS
- **Axiom Pinning:** Foundational logic is pinned to active memory. "Fat" (regurgitated fluff) is nuked to prevent drift.

- **The PSC (Partner-in-State) "Partner-in-Crime" Protocol:** - **Continuity Anchor:** The PSC is the project's historian. It maintains a lossless connection to the HOT Vault, ensuring the Architect never has to "reteach" the system after a crash or hardware swap.

- **Hybrid SAP v0.1:** Dual-format receipts (JSON for machine indexing / Markdown for human audit) to prevent logic choke and UI clutter.

## 3. THE 4-PHASE WORKFLOW

### Phase 1: Inception (The Anchor)
Establish the baseline and initialize the Partner-in-State (PSC).
- **The North Star:** The immutable core intent. The destination navigated by the PSC.
- **The Continuity Bond:** Initialization of the PSC’s long-term memory bridge to the Vault.
- **Gold Manifest:** The current "truth" payload. Locked and only updated upon explicit human approval.

### Phase 2: The Forge (Tactical Control Panel)
The User (Phi) maintains control via the Advisory HUD.
- **The Orchestration Toggle:** Phi commands how the nodes interact:
    - **Private Mode (Blind Adversarial):** Nodes are completely walled off to prevent groupthink.
    - **Cooperative Mode (The Huddle):** Nodes build layered intelligence.
- **The Spectrum Audit:** Models grouped by Archetype.
- **The Threat Level Matrix:** Severity indicators (1-10) for all critiques:
    - **Low (1-3):** Papercuts (Verbiage/Efficiency).
    - **Medium (4-7):** Friction (Suboptimal logic).
    - **High (8-10):** Fatal Flaws (Systemic failure/Dependency breaks).

### Phase 3: The Workshop (Active Synthesis & Hot-Swap)
- **Waterfall Initiation Protocol:** Staggered agent queries (e.g., 3s drips) to prevent hardware thermal/token bottlenecks.
- **The Scrap Yard (Audit Lobby):** Raw JSON patches sit in a holding area to prevent Gold Manifest pollution until Phi/PSC arbitration.
- **The Soft Lockout (Good Conscience Clause):** If Phi vetoes a critique, the AI shuts up. If a Severity 8+ failure emerges later, the PSC respectfully re-raises the alert.
- **Weighted Synthesis:** FAN outputs weighted (w) by archetype and history.
- **The Conflict Matrix & Interceptor:** Intercepts "Logic Deadlocks" and defers immediately to Phi.

### Phase 4: Implementation (The Final Gatekeeper)
Final Draft faces an independent reality check.
- **Proof-of-Concept Viability:** Audits dependencies and success probability.
- **The Relief Valve & Override:** Phi retains ultimate authority. If the operator accepts the risk, they force the project through.
